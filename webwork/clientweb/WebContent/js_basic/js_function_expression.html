함수의 표현식

함수를 선언하고 사용할 수 있지만 표현식으로 정의하는 것을 권고 표현식으로 정의하면 함수의 호이스팅이 일어나지 않는다.
[형식] - 이름이 있는 함수의 표현식
var 변수명 = function 함수명(매개변수 list){
//실행할 코드
}
[= 연산자를 이용해서 함수를 할당하는 표현]
함수가 할당된 변수가 마치 함수의 이름처럼 쓰여짐 따라서 원래 함수의 이름은 사용할 수 없다.





[형식] - 이름이 없는 함수의 표현식
