2. html
   1) 기본태그
      h1~h6, p, br, b,i,font, img, a
      
   2) <img>
      - 이미지를 웹 페이지에 추가하기 위해서 사용
      [문법]
      <img src="이미지의 경로" width="이미지의 가로 사이즈" height="이미지의 세로 사이즈">
      
      [이미지 경로]
       - 파일탐색기에서 확인할 수 있는 파일의 경로를 사용하면 안된다.
         C:\bigdata_iot\work\webwork\clientweb\WebContent\images   ----X
       - 웹에서 요청하는 방식을 이용 
      ① 절대경로
        - 경로를 모두 명시
          <img src="http://192.168.0.221:8088/clientweb/images/view.jpg"/>  
                   ---------------------------
                         root상태
                           
          <img src="/clientweb/images/view.jpg"/>
                   ---
                    root에서 시작하라는 의미
      ② 상대경로
        <img src="clientweb/images/view.jpg"/>
        현재 작업 위치에서 시작하라는 의미
       
        . => 현재위치
        <img src="./clientweb/images/view.jpg"/>
   
        .. => 상위디렉토리 
        <img src="../images/view.jpg"/>

      순서가 없는 목록, 순서가 있는 목록
      
   4) 테이블관련태그
   5) 양식태그
      - 서버와 통신하기 위해서 사용하는 태그(서버로 클라이언트가 입력하거나 선택한 내용을 전송하기 위해서 사용)
      - 무조건 <form></form>사이에 정의해야 한다.(그래야 서버로 데이터가 전송)
      - 서버로 데이터를 전송하기 위해서 값을 구분할 수 있도록 반드시 name속성을 정의해야 한다.
      - 양식태그의 종류 
       * 텍스트관련 양식태그
       * 버튼
         submit - 이 버튼을 누르면 <form></form>사이의 모든 양식 태그를 통해 클라이언트가 입력하거나 선택한 값이 서버로 전송된다.
                  전송되는 데이터의 형식은 name=value
                                    ------ ------
                                    태그 안에     ^
                                    name속성으로  |__사용자가 입력한 값 
                                    정의한 이름    
       * 선택기능
